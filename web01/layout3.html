구글 지도

구글지도 퍼가기 하여 표시하기

  1. 구글 지도를 엽니다.
  2. 구글 지도에서 원하는 위치를 검색하거나 클릭합니다.
  3. 왼쪽 사이드 메뉴에서 공유버튼을 클릭합니다.
  4. 공유 창에서 지도 퍼가기 탭 버튼을 누릅니다.
  5. 지도 퍼가기의 html의 iframe 태그의 오른쪽에 html 복사버튼을 누릅니다.
  6. 지도를 넣고자하는 html 문서에 붙여넣기 합니다.
